# Jo Go Turbo Tables

Single-file browser prototype for a Grade 4-7 multiplication fluency racing game.

## Open The Game

Open `JoGo_Turbo_Tables.html` in any browser.

## Project Contents

- `JoGo_Turbo_Tables.html` - the playable game.
- `screenshots/` - desktop and mobile visual QA screenshots.

## Notes

- Runs offline.
- Uses vanilla HTML, CSS, JavaScript, and canvas.
- No external libraries, accounts, databases, servers, or frameworks.
- Built from the Dividend Dash racing-gate prototype structure, converted to multiplication facts from 1x1 to 12x12.
